2 Core Features

Feed Wall

Latest uploads, trending resources, filtered by tags, infinite scroll pagination

Search & Discovery

Search with filters (subject, course, year level, file type), sort by newest/downloads/ratings

Upload

File picker for PDF, PNG, DOCX, XLSX with metadata form and Cloudinary integration

Notifications

New uploads in followed subjects, download milestones, system announcements

Profile

User stats, uploads history, saved resources, follow subjects

4 API Endpoints

Auth

POST /api/auth/register - Register user

POST /api/auth/login - Login (Firebase token)

GET /api/auth/me - Get current user

Resources

GET /api/resources - Get feed (with filters)

GET /api/resources/:id - Get single resource

POST /api/resources - Upload new resource

PUT /api/resources/:id - Update resource

DELETE /api/resources/:id - Delete resource

GET /api/resources/search - Search resources

POST /api/resources/:id/download - Track download

User

GET /api/users/:id - Get user profile

PUT /api/users/:id - Update profile

GET /api/users/:id/uploads - Get user uploads

POST /api/users/follow-subject - Follow subject

Notifications

GET /api/notifications - Get user notifications

PUT /api/notifications/:id/read - Mark as read

DELETE /api/notifications/:id - Delete notification

6 Setup Steps

1. Cloudinary Setup

  • Sign up at cloudinary.com
  • Get: Cloud Name, API Key, API Secret
  • Enable upload presets (optional)

2. Firebase Setup

  • Create project at console.firebase.google.com
  • Enable Email/Password authentication
  • Get config object for React Native

3. MongoDB Setup

  • Sign up at mongodb.com/atlas
  • Create cluster
  • Get connection string

4. Backend Setup

mkdir studyvault-backend

cd studyvault-backend

npm init -y

npm install express mongoose cors dotenv cloudinary multer firebase-admin

5. Frontend Setup

npx react-native init StudyVault

cd StudyVault

npm install nativewind tailwindcss @react-navigation/native

@react-navigation/bottom-tabs @react-navigation/native-stack

react-native-document-picker react-native-firebase axios zustand
